# Images Asset Folder

This folder contains all the image assets for the FAJI website.

## Directory Structure

- **logos/**: Contains official FAJI logos and partner logos.
  - `logo.png`: Main website logo.
- **sponsors/**: Contains sponsor logos.
- **content/**: Contains general content images (banners, backgrounds, article images).

## Usage

Reference these images in your code starting with `/images/`.

Example:
```tsx
<Image src="/images/logos/logo.png" alt="FAJI Logo" ... />
```
