# Favicon Assets

This folder contains the favicon assets for the FAJI website.

## Instructions

1. Replace the placeholder files in this directory with your actual favicon files.
2. Ensure the filenames match the ones listed below:
   - `android-chrome-192x192.png`
   - `android-chrome-512x512.png`
   - `apple-touch-icon.png`
   - `favicon-16x16.png`
   - `favicon-32x32.png`
   - `favicon.ico`
   - `site.webmanifest` (optional, as Next.js generates one via `manifest.ts`)

You can generate these files using [RealFaviconGenerator.net](https://realfavicongenerator.net/) by uploading your high-resolution logo.
